# Dune-Online
Появилась идея создать с нуля браузерную стратегию по мотивам Дюна битва за Аракис.
Пока есть только заготовки, техники, зданий, стоимости и так далее.

The idea was to create from scratch a browser strategy based on the Battle of Arakis Dune.
So far, there are only blanks, equipment, buildings, cost, and so on.
